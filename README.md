# repomon

scans your git repos in parallel and builds a pre-formatted, color-coded cache for the tmux session switcher. sessions are sorted by most recently used and colored by health — green, yellow, or red. the switcher reads the cache file and pipes it straight to fzf, so it opens instantly.

## setup

```bash
go install github.com/kungfusheep/repomon@latest

# discover repos (writes to ~/.config/repomon/config.json)
repomon --discover ~/code

# build the cache for the first time
repomon --cache

# install background refresh (launchd, every 5 minutes)
repomon --install-cron
```

the tmux session switcher (`~/scripts/tmux_session_switch.sh`) reads `~/.config/repomon/cache` and also triggers a background refresh after each use.

for instant refresh on session switch, add to `.tmux.conf`:

```
set-hook -g client-session-changed 'run-shell -b "repomon --cache"'
```

## how it works

```
repomon --discover ~/code    finds git repos, saves paths to config
         ↓
repomon --cache              scans repos + tmux sessions in parallel,
         ↓                   writes colored output to cache file
~/.config/repomon/cache      tab-delimited: key<TAB>ansi-colored display
         ↓
tmux_session_switch.sh       reads cache, pipes to fzf --ansi, switches session
```

the cache refreshes via:
- **launchd** — every 5 minutes (`--install-cron` / `--uninstall-cron`)
- **on use** — the switcher script runs `repomon --cache &` after each selection
- **on switch** — optionally via tmux `client-session-changed` hook

## commands

```
repomon --cache                  build tmux session cache
repomon --discover ~/code [...]  scan directories for git repos
repomon --export                 markdown report to stdout
repomon --export --output FILE   markdown report to file
repomon --install-cron           install launchd job (5 min interval)
repomon --uninstall-cron         remove launchd job
```

## health

- **GREEN** — clean, recent commits
- **AMBER** — a few dirty files, or stale with uncommitted work
- **RED** — 10+ dirty files, build failures, never committed, or missing

## config

`~/.config/repomon/config.json` — generated by `--discover`, editable by hand.

## files

| path | purpose |
|------|---------|
| `~/.config/repomon/config.json` | tracked repo paths |
| `~/.config/repomon/cache` | pre-built fzf input (ansi colored) |
| `~/.config/repomon/repomon.log` | launchd output |
| `~/scripts/tmux_session_switch.sh` | tmux switcher (reads cache) |
